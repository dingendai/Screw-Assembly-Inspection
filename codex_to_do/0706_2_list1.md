## 任務目標

在系統相機設定中，讓每台檢測相機可以選擇：

1. `原廠自動焦距`
   - 程式完全不控制焦距。
   - 不呼叫 `cv2.CAP_PROP_AUTOFOCUS`。
   - 不呼叫 `cv2.CAP_PROP_FOCUS`。

2. `手動固定焦距`
   - 程式在相機開啟後關閉自動對焦。
   - 程式套用固定焦距值。
   - 焦距值由相機設定頁保存。

第一版只做可設定、可保存、可套用。不要做自動找最佳焦距、清晰度評分或 ROI 校正。

## 成功標準

完成後應滿足：

1. 舊版 `app_config.json` 沒有焦距欄位時，系統仍可正常啟動。
2. 每台 `CameraConfig` 都有 `focus_mode` 與 `manual_focus_value`。
3. 桌面 GUI 相機設定頁可設定焦距模式與固定焦距值。
4. Web 相機設定頁可設定並保存同一組焦距設定。
5. 桌面與 Web 不會互相覆蓋或洗掉焦距設定。
6. `focus_mode == "auto"` 時，程式不控制相機焦距。
7. `focus_mode == "manual"` 時，實體相機開啟後會套用固定焦距。
8. 模擬相機模式不套用焦距，不報錯。
9. 相機不支援手動焦距時，監視流程仍可運作。
10. `python -m compileall app/src` 通過。

## 不做項目

本清單不要做：

1. 不改 `0.py`。
2. 不做 Laplacian 清晰度分數。
3. 不做最佳焦距自動搜尋。
4. 不做 ROI 焦距校正。
5. 不做焦距操作紀錄。
6. 不改 `inference_router.py`。
7. 不改 `qc_db.py`。
8. 不改歷史紀錄與品管統計邏輯。
9. 不控制 OpenCV 的 `CAP_PROP_ZOOM`。

## 建議欄位

在 `CameraConfig` 新增：

```python
focus_mode: str = "auto"
manual_focus_value: int = 120
```

允許值：

```text
focus_mode = "auto" | "manual"
manual_focus_value = 0..255
```

預設值必須是：

```text
focus_mode = "auto"
manual_focus_value = 120
```

原因：避免升級後對既有相機突然關閉自動對焦。

## 注意事項

1. `auto` 模式是「程式不控制」，不是「程式開啟 autofocus」。
2. 不要把欄位命名為 zoom，因為目前 OpenCV 控制的是 focus。
3. 不要讓焦距套用失敗導致相機無法顯示。
4. 不要改動推論、資料庫、歷史紀錄。
5. 新增欄位要保護舊設定檔相容性。
6. 桌面與 Web 必須同步，否則設定會互相覆蓋。
